# Angler

A discord bot to keep track of `angle.wtf` scores.

# Commands

## `!stats` [@user]

Display user stats similar to angle page.

```
85.7 Win%
14 Played
1 Current Streak
3 Max Streak
```

## `!transportador` [@user]

Count the number of times an user guessed in one try.

## `!standings`

Show current standings based on scoring.
